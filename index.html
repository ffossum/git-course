Basic Git usage

init

Initialize your Git repository!

$ git init MyStuff

This creates a .git-folder in a new (or existing) MyStuff-project, containing

  • Configuration
  • All project (meta)data

status

What's happening in my workspace?

$ git status


$ git status
On branch master

Initial commit

nothing to commit (create/copy files and use "git add" to track)

status

$ echo "Content" > MyFirstFile.txt


$ git status
On branch master

Initial commit

Untracked files:
  (use "git add <file>..." to include in what will be committed)

    MyFirstFile.txt

nothing added to commit but untracked files present (use "git add" to track)

add

Adding new or changed files to the commit (staging)

$ git add MyFirstFile.txt


$ git status
On branch master

Initial commit

Changes to be committed:
  (use "git rm --cached <file>..." to unstage)

    new file:   MyFirstFile.txt

rm

Removing staged changes from the commit

$ git rm --cached MyFirstFile.txt


$ git rm --cached MyFirstFile.txt
rm 'MyFirstFile.txt'

$ git status
On branch master

Initial commit

Untracked files:
(use "git add <file>..." to include in what will be committed)

MyFirstFile.txt

nothing added to commit but untracked files present (use "git add" to track)

rm

Removing staged changes completely

$ git rm --force MyFirstFile.txt


$ git rm --force MyFirstFile.txt
rm 'MyFirstFile.txt'
